# Heart Disease Prediction 

## Problem
Predict whether a patient has 
heart disease or not using 
medical data

## Dataset
- 5050 patients
- 18 features including age, BMI,
  blood pressure, cholesterol,
  smoking, stress level etc.

## Steps
- Data Cleaning (NaN, Duplicates, Outliers)
- Encoding (gender, country)
- Feature Engineering
- 3 ML Models compared

## Results
| Algorithm | Accuracy |
|-----------|----------|
| Logistic Regression | 60.09% |
| Decision Tree | 53.80% |
| Random Forest | 59.32% |

## Libraries
- Pandas, Scikit-learn, Matplotlib
